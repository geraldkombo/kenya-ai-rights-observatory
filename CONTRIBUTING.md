# Contributing to the Observatory

We welcome data contributions, particularly regarding unmapped CCTV networks, local internet shutdowns, or automated decision-making systems deployed by county governments.

## How to Contribute Data
1. Fork the repository.
2. Update `src/lib/data.ts` with your verified data.
3. Ensure sources are cited in the commit message (e.g., FOIA request, news report, ODPC filing).
4. Submit a Pull Request.

Please adhere to our primary directive: do not expose PII (Personally Identifiable Information). All metrics must be aggregated at the county level.
